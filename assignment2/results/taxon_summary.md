# Объединяющий таксон

- Ближайший общий таксон (лат.): **Primates**
- Название таксона (рус.): **приматы**
- Таксономический ранг: **отряд**

## Краткое описание

Таксон Primates является ближайшим общим таксоном для выбранных видов, полученных по результатам BLAST. Он объединяет все исследованные последовательности и отражает их эволюционное родство.

Виды, использованные для определения таксона:
- Cercocebus atys (Чёрный мангабей)
- Colobus angolensis (Colobus angolensis)
- Gorilla gorilla (западная горилла)
- Hylobates moloch (Серебристый гиббон)
- Macaca fascicularis (макак-крабоед)
- Mandrillus leucophaeus (Дрил)
- Pan paniscus (карликовый шимпанзе)
- Piliocolobus tephrosceles (Piliocolobus tephrosceles)
- Pongo pygmaeus (Борнейский орангутан)
- Trachypithecus francoisi (Тонкинский гульман)
